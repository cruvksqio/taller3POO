- Descripción del Proyecto: 
Este proyecto es una solución de software desarrollada en Java para la gestión de proyectos de el equipo de desarrollo de software de la empresa TaskForge
El sistema permite la administración de proyectos de desarrollo de software, facilitando la asignación de tareas, el control de estados y la generación de reportes ejecutivos.

El desarrollo se centra en la aplicación estricta de principios de Programación Orientada a Objetos (POO) y Patrones de Diseño, asegurando un código escalable, mantenible y modular.
